Definition of decision table input - contains index (order), mapping to the source variable or source expression, and optional list of allowed values

public class DmnDecisionTableInput

Inheritance: object

Remarks

Source variable and source expression are mutually exclusive and one of them has to be provided. When the definition contains both variable and expression, the expression is used. See CreateDecisionTable(DecisionTable, string, List<net.adamec.lib.common.dmn.engine.engine.definition.IDmnVariable>, List<net.adamec.lib.common.dmn.engine.engine.decisions.IDmnDecision>) for details.
