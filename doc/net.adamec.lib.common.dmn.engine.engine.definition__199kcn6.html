DmnDefinitionFactory Class

Namespace: net.adamec.lib.common.dmn.engine.engine.definition
Assembly: net.adamec.lib.common.dmn.engine
Sources: engine\definition\DmnDefinitionFactory.cs

Factory that validates and transforms the DmnModel to DmnDefinition that can be executed within runtime.DmnExecutionContext

public class DmnDefinitionFactory

Inheritance: object

Remarks

As the DmnParser is responsible for parsing the XML with the model, the DmnDefinitionFactory contains the main logic of checking the model and preparing it for the execution. It's recommended not to build DmnDefinition out of this factory, as the crucial validations may be missed.
